# Pleasure-Way Firefly Interface

Raspberry Pi 3B+ with PiCAN2 SMPS CAN-Bus HAT connected to the Firefly Integrations Vegatouch RV-C network in the 2018 Pleasure-Way Lexor FL. The eRVin software (myervin.com) provides a web dashboard for monitoring and controlling coach systems — tanks, battery, lights, generator, awning, water pump, shore power, and inverter.
